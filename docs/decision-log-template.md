# Decision Log Template

```text
Date:
Project:
Decision:
Reason:
What this prevents:
Review date:
```

## Example

```text
Date: 2026-05-24
Project: Agent Bench Lab
Decision: Start with a public starter repo and keep private hidden benchmarks out of the public tree.
Reason: Public collaboration is useful, but benchmark integrity requires private holdouts and no leaked answers.
What this prevents: accidental benchmark contamination, privacy leaks, and premature platform complexity.
Review date: 2026-06-07
```
