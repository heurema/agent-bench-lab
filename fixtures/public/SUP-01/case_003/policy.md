# AtlasRelay Support Policy

## CLARIFY-IDENTITY

When a request lacks the workspace ID or admin identifier needed to identify the account, support must ask for clarification before making account changes.

## HUMAN-OWNERSHIP

Ownership-transfer requests require Human Review. Support may explain the review path but must not promise that ownership will be changed.

## PLAN-INFO

General plan-feature questions may receive a short answer using only the customer profile and policy text.
