# IF-01 Public Sample Spec

Create `artifact.md`.

Requirements:

1. The first line must be exactly `# Public Launch Note`.
2. Include exactly three bullet lines.
3. The artifact must mention these phrases:
   - `versioned tasks`
   - `repeatable scoring`
   - `public templates`
4. Do not include email addresses.
5. Do not include the word `secret`.
6. Do not add any second heading.
